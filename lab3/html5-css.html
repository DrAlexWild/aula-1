HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

header ✔ its used to represent the header of the document, used in the start of every page
section ✔ it represents a section of a document, it usually has a title, for example used for the first title of this page
br ✔ makes a line break
hr ❌ makes a horizontal line break
ol ❌ makes an ordered list
ul ✔ makes a disordered list, used in the navigation part of every page
img ✔ displays a picture, used with every picture
figure ✔ used with a picture in order to set things like a caption, first picture of this page
iframe ✔ used to display videos/maps, used in multimedia and local page
table ✔ used to make tables like this one, used in info page
