A Internet e a Web

Unidade Curricular Programação web abordaremos os principios basicos associados ao desenvolvimento para a Web, tanto do lado back-end como front-end .

Aprender-se-ão as seguintes tecnologias

  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratorio semanal, de entrega obrigatória, de aplicação da matéria lecionada, Irá ser desenvolvido também um projeto.
